Courses

Significant Courses taken During Undergraduate Degree

  • Data Structures, Algorithms, & Analysis Introducuctory course about complex and fundamental data structures such as arrays, trees, hash tables, graphs. And exploring algorithms for searching and sorting.
  • Introduction to Software Engineering Introduction to the development/evolution of software and the techniques used in the different phases of the software lifecycle.
  • Principles of Operating Systems An introductory course to operating systems principles. Performance measurement; concurrent programs; the management of information, memory and processor resources.
  • Programming Paradigms Course focused on the principles of different programming language paradigms. The three languages we looked at were Haskell, Prolog and Java. Within these languages we explored topics such as data types, lazy evaluation, information hiding and each of their different interpretation principles.
  • Design and Analysis of Algorithms I Course was based around the different techniques for the analysizing algorithms and designing efficient algorithms. For design we focused on greedy methods, divide and conquer and dynamic programming
  • Software Entrepreneurship Development of business models, building software prototypes and creation of pitch presentations to create a software-based business.
  • Web Based Systems An overview of software engineering methods and technologies for developing web-based software systems.
  • Computer Networks Course about the key principles and practice in modern telecommunications and networks. Such as communication protocols, data links, network and internet protocol layers. Also circuit and packet switching.
  • Human Computer Interaction This course focused on the fundamental theory and practice of the design, implementation, and evaluation of human-computer interfaces. This included things like principles of design; methods for evaluating interfaces with or without user involvement; and prototyping.
